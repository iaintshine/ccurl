# ccurl

`ccurl` (aka Color cURL) is a command line utility, wrapper over standard `curl` command with some additional features:

* line numbers
* color output
* HTTP 1.1 syntax highlighting

## Installation

    $ gem install ccurl

## Usage

Use it as a normal `curl` command line tool.
